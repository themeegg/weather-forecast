== Description ==

This is Weather Forecast widget plugin, You can add widget by installing this plugin.
After Installing this plugin go to Appearence>>Widget and add widget to your sidebar.

== Installation ==

Download Weather Forecast plugin and follow these setps.

1. Upload `weather-forecast` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add Weather Forecast widget to your sidebar.
